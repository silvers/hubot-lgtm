hubot-lgtm
==========

A hubot script to get random images from www.lgtm.in

Installation
------------


* Run the `npm install` command.

```
$ npm install hubot-lgtm --save
```

* Add the following code in your `external-scripts.json` file.

```
["hubot-lgtm"]
```

Usage
-----

```
hubot lgtm
```
